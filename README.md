# NLP-Based Clinical Trial Text Analysis

##  Project Description
This project analyzes clinical trial text using Natural Language Processing (NLP) and Machine Learning.  
It takes CSV input and predicts results using trained models.

---

## Features
- Upload CSV file  
- Text preprocessing using NLTK  
- Feature extraction using ELECTRA  
- Prediction using ML models  
- Web interface using Flask  

---

## Technologies Used
- Python  
- Flask  
- NLTK  
- Scikit-learn  
- Transformers (ELECTRA)  

---

## How to Run
1. Install required packages  
2. Run the command:
   python app.py  
3. Open browser:
   http://127.0.0.1:5000
